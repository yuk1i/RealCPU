# RealCPU Project

Make a MIPS CPU and write an operating system.

## Goals

- [x] Single Cycle
- [ ] Memory Access not only for lw/sw
- [ ] Interrupt & Syscalls support
- [ ] L1 Data/Instruction Cache & Unified Memory
- [ ] Initialize $sp and $gp, support $gp addressing
- [ ] Pipeline
- [ ] DDR3 Support
- [ ] SD Card Support
- [ ] Bootloader
